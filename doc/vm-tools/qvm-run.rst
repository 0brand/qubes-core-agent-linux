=======
qvm-run
=======

NAME
====
qvm-run - run a specified command in a specified VM

SYNOPSIS
========
| qvm-run vmname command [aguments]

OPTIONS
=======
--dispvm
    Pass this option instead of vmname to start new DisposableVM
